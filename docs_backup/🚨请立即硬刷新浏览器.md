## 🔍 为什么会这样？

### 浏览器缓存机制

浏览器会缓存 JavaScript 文件以提高加载速度。当我更新代码后：

1. ✅ 代码已经更新到服务器
2. ✅ Git 已经提交更改
3. ❌ 但您的浏览器还在使用旧的缓存文件

### 普通刷新 vs 硬刷新

- **普通刷新** (F5 或 Ctrl+R): 可能使用缓存
- **硬刷新** (Ctrl+Shift+R): 强制重新下载所有文件
