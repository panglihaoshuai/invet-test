## ⚡ 快速开始（2 步）

### 步骤 1: 配置 RESEND_API_KEY

1. 访问 https://resend.com/ 注册
2. 创建 API Key
3. 在 Supabase Dashboard 添加 Secret
   - Name: `RESEND_API_KEY`
   - Value: 您的 API Key

详细说明请查看：**🔧配置RESEND_API_KEY.md**

### 步骤 2: 刷新浏览器

- Windows/Linux: 按 `Ctrl + Shift + R`
- Mac: 按 `Cmd + Shift + R`
