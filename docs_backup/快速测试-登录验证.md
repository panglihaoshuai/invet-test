## 🐛 常见问题

### Q: 收不到验证码邮件？

**解决方案**:
1. 检查垃圾邮件箱
2. 等待1-2分钟（邮件可能延迟）
3. 检查邮箱地址是否正确
4. 确认没有达到发送限制（每小时3封）

### Q: 验证码错误？

**解决方案**:
1. 确认输入的是6位数字
2. 检查是否使用了最新的验证码
3. 确认验证码未过期（5分钟内）

### Q: 登录后没有管理员权限？

2. 刷新浏览器页面
3. 如果仍然没有权限，查看 Supabase Dashboard 中的 profiles 表
